Read more
This research explores whether "spectral filtering," a technique using aggregate structural tests, can accelerate "continuous subgraph matching" (CSM) in dynamic graphs. Subgraph matching is vital for finding patterns in evolving data, such as in bioinformatics or social networks. While spectral filtering effectively prunes candidates in static graphs, its direct application to dynamic graphs with lazy maintenance proved ineffective. The study found that by selectively recomputing local spectra for small neighborhoods, exact local spectral information can be maintained efficiently, even with frequent updates. Integrating this dynamic spectral index into a CSM benchmark demonstrated significant performance gains, removing up to 51% of candidate matches or safely skipping up to 47% of update enumerations. This advancement matters by offering a practical method to improve the efficiency of real-time graph analysis, which is increasingly important as data graphs become more dynamic and complex across various scientific and industrial applications.

SAFARI: Scaling Long Horizon Agentic Fault Attribution via Active Investigation

arXiv:2606.24626v1 Announce Type: new Abstract: As autonomous agents tackle increasingly complex multi-step, multi-agent tasks, their execution trajectories have scaled beyond the constraints of even the largest context windows. Current methods for effectively diagnosing agent failures load the full trajectory into …

Read more
Researchers have developed SAFARI (Scaling long-horizon Agentic Fault AttRibution via active Investigation), a framework designed to diagnose failures in complex, multi-step, multi-agent AI systems. Traditional methods for fault attribution often involve loading an entire execution trajectory into a Large Language Model's (LLM) context window, which becomes unfeasible and inefficient as trajectories grow longer, leading to "attention dilution" and exceeding context limits. SAFARI addresses this by replacing linear context loading with a tool-augmented diagnostic loop. It equips LLMs with specialized tools to read and search specific segments of the trajectory and incorporates a persistent Short-Term Memory (STM) for cross-turn reasoning. This approach effectively decouples diagnostic accuracy from the LLM's architectural context limits, significantly improving performance on long trajectories. This matters because robust fault attribution is essential for debugging, improving reliability, and ensuring the safety of increasingly complex autonomous AI agents in real-world deployments.

RIFT-Bench: Dynamic Red-teaming For Agentic AI Systems

cs.AI updates on arXiv.orgJun 24, 13:00 KST

arXiv:2606.23927v1 Announce Type: new Abstract: Agentic AI systems powered by large language models (LLMs) are rapidly evolving into autonomous decision-making systems, exposing attack vectors beyond those of traditional LLM vulnerabilities. Existing security evaluations are often tied to specific implementations or …

Read more
Researchers have introduced RIFT-Bench, a novel methodology for "dynamic red-teaming" of agentic AI systems. Agentic AI systems, which are autonomous decision-making entities powered by Large Language Models (LLMs), present new security vulnerabilities beyond those of traditional LLMs. Existing security evaluations often lack a unified approach, being tied to specific implementations or domains. RIFT-Bench addresses this by using a graph representation to enable consistent evaluations across diverse agentic architectures. It operates in two automated phases: "Discovery," which extracts the system's structure, and "Scanning," which deploys adaptive adversarial attacks. The system then generates a comprehensive evaluation report based on dynamically adaptable adversarial probes across various attack vectors and objectives. This development is crucial because it provides a standardized, automated framework for identifying and assessing security weaknesses in complex autonomous AI systems, which is vital for developing more secure and robust AI agents before their deployment in critical applications.
